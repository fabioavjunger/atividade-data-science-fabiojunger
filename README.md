# Projeto de Ciência de Dados - Análise Exploratória do Dataset IMDB

## Objetivo

Este repositório foi criado para a Atividade 2 da disciplina Técnicas e Algoritmos em Ciência de Dados. O projeto apresenta uma análise exploratória de dados (EDA) utilizando um dataset do IMDB, com foco em tratamento de valores ausentes, visualização de distribuições, correlação entre variáveis numéricas e análise de ROI dos filmes.

## Arquivos do Projeto

* `Cópia de Atividade 1_ Guia de Análise Exploratória de Dados (EDA) - Dataset IMDB.ipynb`: notebook com toda a análise desenvolvida no Google Colab
* `requirements.txt`: bibliotecas utilizadas no projeto
* `.gitignore`: arquivos e pastas ignorados no versionamento

## Como executar localmente

1. Clone o repositório:

```bash
git clone https://github.com/fabioavjunger/atividade-data-science-fabiojunger.git
```

2. Acesse a pasta do projeto:

```bash
cd atividade-data-science-fabiojunger
```

3. Instale as dependências:

```bash
pip install -r requirements.txt
```

4. Abra o notebook em Jupyter Notebook, JupyterLab ou VS Code.
