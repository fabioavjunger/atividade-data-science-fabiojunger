# Projeto de Ciência de Dados - Heart Disease

## 📊 Objetivo

Este projeto tem como objetivo realizar uma análise completa de dados relacionados a doenças cardíacas, incluindo exploração, tratamento e modelagem preditiva.

## 📁 Estrutura do Projeto

```
atividade_data_science/
├── data/
│   ├── raw/
│   └── processed/
├── notebooks/
├── README.md
├── requirements.txt
```

## 🚀 Como executar

1. Clone o repositório:

```
git clone https://github.com/fabioavjunger/atividade-data-science-fabiojunger.git
```

2. Instale as dependências:

```
pip install -r requirements.txt
```

3. Execute os notebooks na pasta notebooks/

## 📊 Dataset

O projeto utiliza dados de doenças cardíacas com múltiplas variáveis clínicas.

## 🛠 Tecnologias utilizadas

* Python
* Pandas
* Seaborn
* Matplotlib
* Scikit-learn
